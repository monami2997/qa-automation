# VS Code Keyboard Shortcuts for Windows

This document contains a comprehensive list of commonly used keyboard shortcuts in Visual Studio Code for Windows.

## General Shortcuts

| Shortcut | Description |
|----------|-------------|
| `Ctrl + Shift + P` | Open Command Palette |
| `Ctrl + P` | Quick Open, Go to File |
| `Ctrl + Shift + N` | New Window/Instance |
| `Ctrl + Shift + W` | Close Window/Instance |
| `Ctrl + ,` | Open User Settings |
| `Ctrl + K Ctrl + S` | Open Keyboard Shortcuts |
| `Ctrl + Shift + X` | Open Extensions |
| `Ctrl + Shift + G` | Open Source Control |
| `Ctrl + Shift + D` | Open Debug |
| `Ctrl + Shift + F` | Open Search |
| `Ctrl + Shift + H` | Replace in Files |
| `Ctrl + Shift + J` | Toggle Panel |
| `Ctrl + Shift + U` | Show Output Panel |
| `Ctrl + Shift + Y` | Show Debug Console |
| `F11` | Toggle Full Screen |
| `Ctrl + =` | Zoom In |
| `Ctrl + -` | Zoom Out |
| `Ctrl + Shift + P` then "reload" | Reload Window |

| `Ctrl + Shift + Enter` | Insert Line Above |
| `Alt + Up/Down` | Move Line Up/Down |
| `Shift + Alt + Up/Down` | Copy Line Up/Down |
| `Ctrl + Shift + L` | Select All Occurrences of Current Selection |
| `Ctrl + D` | Add Selection to Next Find Match |
| `Ctrl + K Ctrl + D` | Move Last Selection to Next Find Match |
| `Ctrl + F2` | Select All Occurrences of Current Word |
| `F2` | Rename Symbol |
| `Ctrl + .` | Quick Fix |
